W3C

The World Wide Web Consortium is the main international standards organization for the World Wide Web. Founded in 1994 and currently led by Tim Berners-Lee, the consortium is made up of member organizations that maintain full-time staff working together in the development of standards for the World Wide Web.

Its mission is to lead the World Wide Web to its full potential by developing standards, protocols and guidelines that ensure its long-term growth. Using W3C-compliant XML and CSS coding also improves search engine optimization by allowing search engine robots to “crawl” through websites quicker and more efficiently.

HTML

HyperText Markup Language (HTML) is the set of markup symbols or codes inserted into a file intended for display on the Internet. The markup tells web browsers how to display a web page's words and images.

HTML was made so that people do not have to write programs to just show a button, this widget was made by a simple markup, styled with a cascading style sheet and you can generate a good page with just that.

If you take JavaScript to write everything on a page, it will be programmatically intensive will not run with JavaScript disabled and you will end up having more bugs than you could have asked.

So, for very complex web pages people go for JavaScript tools (like react.js) to manipulate the DOM else for simple or static web pages HTML holds the ground for efficiency and hence it is very important.

For example, we can create a paragraph by placing the enclosed text within a starting <p> and closing </p> tag.

CSS

Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in a markup language such as HTML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

CSS helps you to keep the informational content of a document separate from the details of how to display it. The details of how to display the document are known as its style. You keep the style separate from the content so that you can:

  • Avoid duplication.
  • Make maintenance easier.
  • Use the same content with different styles for different purposes.

Want to create a designation for paragraphs that should be in small caps? A CSS sample for that would be:

p.smallcaps { font-variant: small-caps; }
